03 / INTERPRETATION

What the table does not contain.

Not a cold-navigation trace

Activation excludes HTML insertion and JavaScript parse/evaluation. Sizes are reported, but network, FCP and LCP are not yet traced.

Not presentation time

The next-frame metric stops at a requestAnimationFrame callback. A browser trace is still needed to split scripting, style, layout and paint.

Not all of React

Kelta lacks React's async, transition, Suspense, error-boundary and ecosystem surface. This suite tests a narrow shared subset.

Not the final baseline set

A React + incremental external-store variant is still needed to separate renderer cost from derived-data strategy.
